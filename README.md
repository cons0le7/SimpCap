# SimpCap
A very simple packet capture tool. 

-logs are stored in /SimpCap/logs.pcap 

-Requires root.


# Depends 
git 

python3

scapy

# Installation: 

``` 
git clone https://github.com/cons0le7/SimpCap 
cd SimpCap
sudo python3 simpcap.py 
```
